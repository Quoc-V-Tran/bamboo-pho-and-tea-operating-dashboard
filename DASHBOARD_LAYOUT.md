## Key Features

### 1. Tomorrow's Prediction (Top Section)
**Left Column (1/3 width):**
- Interactive inputs for forecast
- Large metric showing predicted bowls

**Right Column (2/3 width):**
- 6 key metrics in 3 columns
- January summary stats
- Model coefficients (Weekend lift, Precip impact, Temp effect)
- Model quality (R¬≤)

### 2. Visualizations
- **Line Chart**: Time series showing relationship between bowls sold and temperature
- **Scatter Plot**: Shows individual days with regression lines for weekend vs midweek

### 3. Loyalty Tracker
- Table format showing top customers
- Sorted by visit frequency
- Shows total spend per customer
