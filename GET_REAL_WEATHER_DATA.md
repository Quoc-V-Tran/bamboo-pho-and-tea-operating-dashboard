## Method 1: NOAA Climate Data Online (Recommended)

### Step-by-Step Instructions

1. **Visit NOAA Climate Data Online:**
   - Go to: https://www.ncdc.noaa.gov/cdo-web/

2. **Search for Weather Data:**
   - Click on "Search Tool" or "Search for Data"
   - Select "Daily Summaries" dataset
   - Click "Continue"

3. **Select Location:**
   - Search for: **"Harrisburg, PA"** or **"KMDT"** (airport code)
   - Or use station: **GHCND:USC00363698** (Harrisburg 1 NE)
   - Click "Add to Cart"

4. **Select Date Range:**
   - Start Date: **01/01/2025**
   - End Date: **12/31/2025**

5. **Select Data Types:**
   - Check: **TMAX** (Maximum Temperature)
   - Check: **PRCP** (Precipitation)
   - Optionally: TMIN, SNOW, SNWD

6. **Output Format:**
   - Select: **CSV** format
   - Select: **Standard** units (Fahrenheit)

7. **Submit Order:**
   - Enter your email address
   - Submit the order
   - You'll receive a download link via email (usually within minutes)

8. **Download the CSV file**

## Method 2: Weather Underground (Alternative)

### Step-by-Step Instructions

1. **Visit Weather Underground History:**
   - Go to: https://www.wunderground.com/history/daily/us/pa/camp-hill

2. **Select Each Month:**
   - Use the month selector to choose each month of 2025
   - Click through January 2025, February 2025, etc.

3. **Copy the Data:**
   - For each month, copy the daily high temperatures
   - Note any precipitation (rain, snow, etc.)

## Method 3: Penn State Climatologist (PA-specific)

### Step-by-Step Instructions

1. **Visit Penn State Climate Data:**
   - Go to: https://climate.met.psu.edu/data

2. **Select Data Archive:**
   - Choose the data archive option
   - Select Harrisburg or nearby station

3. **Download 2025 Data:**
   - Select date range: 2025
   - Download the data file
